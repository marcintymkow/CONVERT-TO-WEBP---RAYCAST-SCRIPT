# Convert to WebP – Raycast Script  A Raycast script for converting images to WebP format with [Squoosh](https://squoosh.app/)-like quality.  ## Features  - Convert JPG, PNG, TIFF, GIF, BMP → WebP - Squoosh-like compression quality (default: 82) - Batch processing – convert multiple files at once - Shows total space saved
